- ### Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/suhas991/20_Simple-React-Projects.git
   ```

2. Install any required dependencies (if applicable):

   ```bash
   cd qrCode
   npm install
   npm i react-qr-code
   ```

3. Run the game:


   ```bash
   npm run dev
   ```
